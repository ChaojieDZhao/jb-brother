## 查看Class文件编译版本
 
> 下载[HexEditor.dll](https://sourceforge.net/projects/npp-plugins/files/Hex%20Editor/)文件放置在$NotePad++\\plugins，重启NotePad++。   
> 插件 >> HEX Editor >> View in Hex，查看第一行的第7个字节进行对比。

|版本号|	对应十进制|jdk版本号|
|-|-|-|
|2E|46|jdk1.2|
|2F|47|jdk1.3|
|30|48|jdk1.4|
|31|49|jdk1.5|
|32|50|jdk1.6|
|33|51|jdk1.7|
|34|52|jdk1.8|
