## 小节和节拍

G#：G sharp，读作升G

Bь：B flat，读作降B

4/4 分子代表每小节有4拍，分母代表4分（之一）音符为1拍。

4/8 分子代表每小节有4拍，分母代表8分（之一）音符为1拍。

6/8 分子代表每小节有6拍，分母代表8分（之一）音符为1拍，这就是常说的86拍。
